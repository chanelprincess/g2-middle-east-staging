# Cloudflare Pages Build Configuration

When setting up the project in Cloudflare Pages dashboard, use these exact settings:

## Build Configuration

**Framework preset:** Next.js

**Build command:**
```bash
npm run build
```

**Build output directory:**
```
.next
```

**Root directory:** (leave empty)

## Environment Variables

Add these in the Cloudflare Pages dashboard under Settings > Environment variables:

```
NODE_VERSION=20
```

## Branch Configuration

- **Production branch:** main
- **Preview branches:** All other branches

## Notes

- Do NOT use `pages:build` or `@cloudflare/next-on-pages` - it's deprecated
- Standard Next.js build works perfectly with Cloudflare Pages
- All 35 pages will be generated as static HTML
- Dynamic routes will work with Cloudflare's edge runtime automatically
